What It Does

Panek Video Program is a free, open-source desktop tool that converts a single static image and an audio file into a professional video ready for upload to any platform.

  • Images: JPG, PNG, WEBP
  • Audio: MP3, WAV
  • Output: H.264 MP4, 1920√ó1080, 30 FPS
  • Quality: CRF 20, AAC audio at 192kbps
  • Smart scaling: Automatic 16:9 letterboxing

How to Use

Create videos in three simple steps:

  1. Choose your files: Select an image and an audio file
  2. Set output: Optionally choose where to save and set a title
  3. Render: Click the Render Video button

The render button turns green when both files are selected. Rendering typically takes 10-30 seconds for a 3-4 minute song.

Native System Integration

Works seamlessly with your desktop environment using native file dialogs:

  • KDE/Plasma: Uses kdialog
  • GNOME/XFCE: Uses zenity
  • macOS: Uses osascript
  • Fallback: Qt-based dialogs

Feels like a native application on every platform.

Requirements

FFmpeg Required: Install via your system's package manager.

Ubuntu/Debian: sudo apt install ffmpeg
Fedora: sudo dnf install ffmpeg
macOS: brew install ffmpeg

Frequently Asked Questions

Does it require FFmpeg?

Yes, FFmpeg and FFprobe must be installed on your system. You can install FFmpeg using your system's package manager.

What if my image isn't 16:9?

The program automatically scales your image to fit 16:9 aspect ratio and adds black letterboxing or pillarboxing as needed. Your image maintains its aspect ratio.

Can I use the videos commercially?

Yes! You own all the videos you create with Panek Video Program and can use them for any purpose, including commercial projects.

How long does rendering take?

Typically 10-30 seconds for a 3-4 minute song, depending on your computer's processing power.

What platforms are supported?

Currently Linux (all major distributions) and macOS. Windows support may come in future versions.
