Panek Video Program

Turn one image and one audio file into a 16:9 MP4 video. Native dialogs. Fast. Simple.

Download (Linux AppImage / .deb) View Source on GitHub

What it does

Produces a clean 1920Ã—1080 MP4 at 30 FPS using your image and audio track.

  • Image: JPG, PNG, WEBP
  • Audio: MP3, WAV
  • Video: H.264 yuv420p, CRF 20

How to use

  1. Open the app and choose an output folder.
  2. Select your image + audio.
  3. Click Render Video.

Render button turns green when both files are selected.

Native Feel

Uses native system dialogs when available:

  • KDE: kdialog
  • GNOME/XFCE: zenity
  • macOS: osascript
  • Fallback: Qt dialogs

FAQ

Needs ffmpeg? Yes, install via system package manager.

Letterboxing? The image is scaled and padded so the output is always 16:9.
