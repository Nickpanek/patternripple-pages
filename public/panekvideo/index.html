Panek Video Program

Turn one image and one audio file into a clean 16:9 MP4. Native file dialogs on Linux and macOS. Fast and simple.

Download - Linux View source on GitHub

What it does

It combines a still image with an audio track and renders a 1920x1080 MP4 at 30 fps with AAC audio and faststart for web upload. The render button turns green when both files are selected.

  • Image formats: JPG, PNG, WEBP
  • Audio formats: MP3, WAV
  • Video: H.264 yuv420p, CRF 20

How it works

  1. Choose an output folder.
  2. Select your image and audio.
  3. Optionally set a title. Click Render Video.

Native picker cascade on Linux and macOS: kdialog, zenity, osascript, then Qt fallback.

Why it looks native

The app prefers platform dialogs first, so file pickers match your desktop. If a native tool is missing, it falls back to Qt.

FAQ

Is ffmpeg required? Yes. Install via your package manager.

Will it letterbox my image? Yes, it scales to fit and pads to 16:9.

Where are files saved? In the output folder you select.
